Spectrum Engine

The Spectrum Engine is a simple collection of classes designed to make the creation of 2D or 3D games easier, building upon SFML 2.

Currently, the engine has been used to create the game "Darkhold: Exile", a game that takes inspiration from the Hexen series.

At the moment, the repo is dependent on several personal code libraries that are not yet included with the source, as well as the GME (Game Music Emulator) source.

Engine use

The engine is free to use under the GPL v2, though a word of warning: It's not going to be easy to use at this point in time. The documentation is still very much 'in-progress'.
